# Research Monthly Review

Scaffold skill only. The repository includes a template, but no working monthly aggregation or threshold-check implementation.

## Workflow

1. Load thresholds and the prior month of daily briefs.
2. Load deep dives completed in the same window.
3. If thresholds are not met, skip formal monthly report generation.
4. Summarize representative papers, topic trends, deep-dive recaps, and next-month recommendations.
5. Write the monthly note into Obsidian.
