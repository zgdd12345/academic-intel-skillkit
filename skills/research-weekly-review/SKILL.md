# Research Weekly Review

Scaffold skill only. The repository includes a template, but no working weekly aggregation or threshold-check implementation.

## Workflow

1. Load thresholds and the prior 7 days of daily briefs.
2. Load deep dives completed in the same window.
3. If thresholds are not met, do not generate a formal weekly report.
4. Summarize top papers, topic shifts, deep dives, and watchlist items.
5. Write the weekly note into Obsidian.
