# Suggested Obsidian Layout

Audience: human readers setting up their vault structure. This file is a usage/reference note, not an agent execution contract.

Only `01_Daily/` is needed for the current MVP. The other folders are optional placeholders if you want a stable structure for later manual or scaffolded workflows.

```text
Research_Intel/
├── 00_Config/
├── 01_Daily/
├── 02_Deep_Dives/
├── 03_Weekly/
├── 04_Monthly/
├── 05_Index/
└── 99_System/
```

Recommended notes:

- MVP path: `01_Daily/YYYY_MM_DD_Daily.md`
- Optional future path: `02_Deep_Dives/YYYY/paper-<id>-summary.md`
- Optional future path: `02_Deep_Dives/YYYY/paper-<id>-repro.md`
- Optional future path: `03_Weekly/YYYY-Www-academic-weekly.md`
- Optional future path: `04_Monthly/YYYY-MM-academic-monthly.md`
